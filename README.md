# ✦ Moment of Wisdom — Quote Generator

A stunning, animated daily quote generator with a **3D Three.js background**, typewriter effect, copy to clipboard, and image download — built with **HTML**, **CSS**, and **JavaScript**.

## 🌐 Live Demo
👉 [Click here to view live](https://memon744.github.io/task-05/)

---

## ✨ Features

- 🌀 **3D Animated Background** — rotating torus knot with floating particles powered by Three.js
- ⌨️ **Typewriter Effect** — quotes animate letter by letter for a premium feel
- 🎨 **Luxury Dark UI** — deep purple gradient with glowing pink & purple orbs
- 📋 **Copy to Clipboard** — copy the full quote + author with one click
- 📥 **Download as Image** — save the quote card as a `.png` file via html2canvas
- 🖱️ **Mouse Parallax** — 3D object reacts to mouse movement
- 🔔 **Toast Notifications** — smooth animated feedback on copy and download actions
- 📱 **Fully Responsive** — stacked layout for mobile screens

## 🛠️ Built With

- **HTML5**
- **CSS3** — Glassmorphism, keyframe animations, radial gradients, backdrop-filter
- **Vanilla JavaScript** — Typewriter engine, clipboard API, toast system
- **[Three.js](https://threejs.org/)** — 3D torus knot + particle system
- **[html2canvas](https://html2canvas.hertzen.com/)** — Quote card image export
- **Google Fonts** — Playfair Display, Nunito

## 🔧 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/memon744/task-05.git
   ```
2. Open `index.html` in any modern browser — done! ✅

> Requires an internet connection to load Three.js, html2canvas, and Google Fonts from CDN.

---

## 🎮 How to Use

| Button | Action |
|--------|--------|
| `✦ New Quote` | Generates a new random quote with typewriter effect |
| `⊕ Copy` | Copies the quote and author to clipboard |
| `↓ Download` | Saves the quote card as a PNG image |
| Mouse Move | Rotates the 3D torus knot interactively |
